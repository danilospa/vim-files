# Danilo's dot files

## Installing Vim Plug dependency manager

```bash
git submodule update --init --recursive
```

## Installing Vim-Plug dependencies

Install [Ack](https://beyondgrep.com/install/).

Run vim and execute the following:
```
:PlugInstall
```

## Setup dot files

Run
```bash
./setup.sh
```
