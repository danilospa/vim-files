# Danilo's dot files

## Setup dot files

Run
```bash
./setup.sh
```

## Installing Vim-Plug dependencies

Install [ripgrep](https://github.com/BurntSushi/ripgrep#installation).
Run vim and execute `:PlugInstall`.

## Installing tmux plugins

On tmux, hit `prefix + I`.
